Selamat datang di kelas dasar HTML. Dalam pembelajaran ini, kamu akan memulai langkah pertamamu untuk memahami bagaimana sebuah website dibangun. HTML, atau HyperText Markup Language, adalah bahasa yang digunakan untuk membuat struktur dasar halaman web.

Bayangkan HTML seperti kerangka pada sebuah bangunan. Tanpa HTML, sebuah situs web tidak akan memiliki bentuk atau susunan yang jelas. Karena itu, memahami HTML adalah langkah pertama yang sangat penting bagi siapa pun yang ingin belajar pemrograman web.

Di bab ini, kamu akan diajak mengenal lebih dekat tentang peran HTML, cara kerjanya, dan mengapa bahasa ini menjadi dasar dari seluruh halaman web yang ada di internet. Tidak perlu khawatir jika kamu belum pernah menulis kode sebelumnya, karena pembelajaran ini dibuat dengan penjelasan yang mudah dimengerti.

Setelah menyelesaikan bab ini, kamu akan memahami apa itu HTML dan mengapa setiap web developer perlu menguasainya. Mari mulai perjalananmu menuju dunia pengembangan web dengan semangat belajar dan rasa ingin tahu yang tinggi.

Sekarang, setelah kamu mengenal HTML secara umum, mari kita pelajari lebih dalam apa sebenarnya HTML itu dan bagaimana cara kerjanya dalam membangun halaman web.

Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML digunakan untuk membuat struktur dasar dari sebuah halaman website. HTML bukan bahasa pemrograman seperti JavaScript atau Python, melainkan bahasa penandaan (markup language) yang memberi tahu browser bagaimana menampilkan isi halaman.

Struktur Dasar HTML

Setiap halaman HTML memiliki struktur dasar yang biasanya seperti ini: 

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>

Mari kita pahami bagian-bagian dari kode di atas:

  • <!DOCTYPE html> – Memberi tahu browser bahwa dokumen ini menggunakan HTML versi terbaru (HTML5).
  • <html> ... </html> – Pembungkus utama dari seluruh dokumen HTML.
  • <head> ... </head> – Berisi informasi tentang halaman seperti judul dan metadata.
  • <body> ... </body> – Berisi semua isi yang akan tampil di layar browser, seperti teks, gambar, dan tautan.

Bagaimana HTML Bekerja?

Browser seperti Chrome atau Firefox membaca file HTML dan menampilkan hasilnya ke layar berdasarkan struktur kode yang kamu tulis. Misalnya, jika kamu menulis tag <h1>, browser tahu bahwa itu adalah judul besar.

Jadi, HTML berperan seperti kerangka dari sebuah rumah. Ia menentukan di mana letak ruang tamu, dapur, dan kamar, sementara bahasa lain seperti CSS akan memberi warna dan dekorasi, dan JavaScript akan menambahkan interaksi.

Setelah kamu mengenal struktur dasar HTML, sekarang kita akan membahas bagian yang paling penting dalam HTML yaitu elemen. Elemen adalah dasar dari semua yang kamu lihat di halaman web. Hampir setiap bagian dari halaman website — mulai dari teks, gambar, tautan, hingga tabel — semuanya adalah elemen HTML.

1. Pengertian Elemen HTML

Elemen HTML adalah blok penyusun utama dalam sebuah halaman web. Setiap elemen biasanya memiliki tag pembuka, isi, dan tag penutup. Tag digunakan untuk memberi tahu browser jenis konten apa yang akan ditampilkan.

<p>Halo, ini adalah sebuah paragraf.</p>

Contoh di atas adalah satu elemen HTML lengkap, yaitu elemen <p> (paragraf). Bagian <p> disebut tag pembuka, teks di tengahnya adalah isi elemen, dan </p> adalah tag penutup.

2. Jenis-jenis Elemen HTML

Ada dua jenis elemen HTML yang penting untuk kamu ketahui:

  • Elemen Blok (Block Element)
    Elemen jenis ini selalu dimulai di baris baru dan menempati seluruh lebar halaman. Contohnya: <div>, <p>, <h1> – <h6>.
  • Elemen Inline (Inline Element)
    Elemen ini tidak membuat baris baru dan hanya mengambil ruang sesuai kebutuhan. Contohnya: <span>, <a>, <strong>.

3. Elemen di Dalam Elemen

HTML memungkinkan kamu untuk meletakkan satu elemen di dalam elemen lainnya. Ini disebut nested element. Misalnya:

<p>Saya suka <strong>belajar HTML</strong> karena seru.</p>

Dalam contoh di atas, elemen <strong> berada di dalam elemen <p>. Hasilnya, teks “belajar HTML” akan ditampilkan dalam huruf tebal.

4. Kesalahan Umum

Dalam menulis HTML, pastikan setiap tag pembuka memiliki tag penutup yang sesuai dan urutannya benar. Misalnya, jangan seperti ini:

<p><strong>Halo Dunia</p></strong>

Contoh di atas salah karena urutan penutupnya tidak sesuai. Penulisan yang benar adalah:

<p><strong>Halo Dunia</strong></p>

Urutan tag harus seperti “lapisan bawang” — tag terakhir dibuka adalah yang pertama ditutup.

Pada bab sebelumnya kamu sudah belajar tentang elemen HTML dan bagaimana strukturnya. Sekarang, kita akan membahas atribut — bagian kecil yang sangat penting di setiap elemen HTML. Atribut membantu kita memberikan informasi tambahan pada elemen agar browser tahu bagaimana cara menampilkannya.

1. Apa Itu Atribut?

Atribut HTML adalah informasi tambahan yang ditulis di dalam tag pembuka suatu elemen. Setiap atribut memiliki dua bagian utama:

  • Nama atribut — menunjukkan jenis informasi yang diberikan.
  • Nilai atribut — berisi isi atau data dari atribut tersebut, ditulis di dalam tanda kutip.
<img src="gambar.jpg" alt="Gambar pemandangan">

Pada contoh di atas:

  • src adalah nama atribut (menunjukkan lokasi gambar).
  • "gambar.jpg" adalah nilai atributnya.
  • alt berfungsi untuk memberikan teks alternatif jika gambar tidak bisa ditampilkan.

2. Cara Menulis Atribut

Penulisan atribut harus berada di dalam tag pembuka elemen. Umumnya ditulis seperti ini:

<elemen namaAtribut="nilai">Isi elemen</elemen>

Beberapa contoh penulisan atribut:

  • <a href="https://www.idnboarding.sch.id">Website IDN</a> – Membuat tautan menuju situs tertentu.
  • <img src="foto.jpg" alt="Foto siswa"> – Menampilkan gambar dan teks alternatif.
  • <input type="text" placeholder="Masukkan nama"> – Membuat kolom input dengan teks petunjuk.

3. Atribut Umum yang Sering Digunakan

Berikut adalah beberapa atribut penting yang sering kamu temui dalam HTML:

Nama Atribut Fungsi Contoh
class Memberikan nama kelas pada elemen (biasanya digunakan di CSS). <div class="kotak"></div>
id Memberikan identitas unik pada elemen. <p id="judul">Selamat Datang</p>
style Menambahkan gaya langsung pada elemen (inline CSS). <h1 style="color:blue;">Halo</h1>
title Menampilkan teks kecil saat kursor diarahkan ke elemen. <button title="Klik untuk kirim">Kirim</button>

4. Atribut Khusus pada Beberapa Elemen

Beberapa elemen HTML memiliki atribut khusus yang hanya bisa digunakan pada elemen tersebut. Contohnya:

  • href pada <a> – untuk tautan.
  • src pada <img> – untuk gambar.
  • type pada <input> – menentukan jenis input (teks, password, tombol, dsb).

5. Tips Menulis Atribut dengan Benar

  • Gunakan tanda kutip ganda (" ") untuk menulis nilai atribut.
  • Tulis nama atribut dalam huruf kecil agar konsisten dan mudah dibaca.
  • Gunakan atribut yang sesuai dengan fungsinya agar struktur HTML tetap rapi.

Dengan memahami atribut HTML, kamu bisa membuat elemen yang lebih informatif dan interaktif. Di bab selanjutnya, kita akan menutup kelas HTML Dasar dengan kesimpulan dan latihan kecil agar kamu makin paham sebelum lanjut ke CSS.

Apa yang Sudah Kamu Pelajari?

  • Elemen dan Tag HTML: Kamu sudah mengenal berbagai elemen penting seperti <h1> untuk judul, <p> untuk paragraf, <a> untuk tautan, dan <img> untuk gambar.
  • Atribut HTML: Kamu tahu bahwa elemen HTML dapat memiliki atribut seperti src, href, dan alt yang memberikan informasi tambahan bagi browser.

Peran HTML dalam Dunia Web

HTML bukan hanya sekadar kode. Ia adalah bahasa yang menjadikan ide dan desain dapat tampil di layar pengguna. Tanpa HTML, tidak akan ada struktur yang bisa ditampilkan oleh CSS maupun dihidupkan oleh JavaScript.

Langkah Selanjutnya

Setelah memahami HTML, langkah berikutnya yang bisa kamu pelajari adalah:

  • CSS (Cascading Style Sheets) untuk memperindah tampilan halaman.
  • JavaScript untuk menambahkan interaktivitas.
  • Version Control (Git & GitHub) untuk mengelola dan menyimpan kode dengan rapi.

Keterampilan dalam HTML adalah pondasi awal untuk menjadi seorang web developer. Dengan konsistensi belajar dan latihan, kamu akan bisa membangun situs yang menarik dan fungsional.

Tips: Cobalah membuat halaman sederhana berisi profil dirimu. Gunakan elemen-elemen yang telah kamu pelajari seperti judul, paragraf, gambar, dan tautan. Ini akan membantu memperkuat pemahamanmu.
